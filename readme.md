# ThinkPlugsAdmin

#### 介绍

基于 ThinkAdmin 的 admin 模块

#### 全新安装 ( 空目录下执行 )

```shell
composer require zoujingli/think-library v6.0.x-dev
composer require zoujingli/think-plugs-admin dev-master
```

#### 增量安装 ( 项目根目录执行 )

```shell
composer require zoujingli/think-plugs-admin dev-master
```

#### 启动测试环境

```shell
php think run --host 127.0.0.1
```
